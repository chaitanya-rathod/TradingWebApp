# Trading Web App - Backend

This is the backend server for the TradingWebApp project. The backend handles business logic, database interactions, and API routes to serve both the frontend and admin dashboard.

## Features

- RESTful API for handling user registration, authentication, and trading operations
- Real-time data updates for trading activities
- Secure data handling and encryption
- Integrated with a PostgreSQL database for data persistence

## Technologies Used

- **Node.js** - JavaScript runtime for building server-side applications
- **Express.js** - Fast, unopinionated, minimalist web framework for Node.js
- **PostgreSQL** - Relational database for data persistence
- **JWT** - For handling user authentication
- **Axios** - HTTP client for making API requests

## Getting Started

### Prerequisites

Ensure you have the following installed on your local machine:

- **Node.js** (version 14 or higher)
- **npm** (Node Package Manager)
- **PostgreSQL** - Make sure you have a running PostgreSQL instance
